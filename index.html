Project Overview

This project transforms a standard laptop into a functional digital oscilloscope. By capturing ambient sound signals through the laptop's built-in microphone, it provides real-time signal analysis capabilities typically reserved for specialized hardware.

The core engine uses Python to process the audio input stream precisely. Utilizing powerful libraries such as NumPy for numerical operations, SoundDevice for low-latency audio capture, and Matplotlib for high-speed rendering, the system displays dynamic waveforms mirroring professional oscilloscope equipment.

import sounddevice as sd
import numpy as np
import matplotlib.pyplot as plt
from matplotlib.animation import FuncAnimation

# Parameters
FS = 44100       # Sample rate
CHUNK = 1024     # Number of frames per buffer
DURATION = 0.05  # Time scale of window

# Setup plot
fig, ax = plt.subplots()
x = np.linspace(0, DURATION, int(FS * DURATION))
line, = ax.plot(x, np.zeros(len(x)), color='#00ff41')

ax.set_ylim(-1, 1)
ax.set_xlim(0, DURATION)
fig.patch.set_facecolor('#0d1117')
ax.set_facecolor('#0d1117')
ax.axis('off')
